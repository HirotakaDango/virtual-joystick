# 🎮 Virtual Joystick Bookmarklet

A sleek, draggable, and resizeable on-screen virtual controller that works on **any website**. Perfect for playing browser games or navigating web pages on mobile devices.

## ✨ Features
* **Auto-Scaling**: Automatically scales to fit any screen size perfectly.
* **Dual Analog Sticks**: Left stick maps to `WASD`, right stick controls a virtual mouse pointer.
* **D-Pad**: Maps to the standard `Arrow` keys.
* **Action Buttons**: Maps to classic gaming keyboard controls.
* **Flat Modern UI**: Clean, flat design with perfectly centered geometric buttons.

## 🛠️ How to Install
1. Create a new bookmark in your browser (right-click your bookmarks bar and select **Add Page** or **Add Bookmark**).
2. Name it something like `🎮 PS Joystick`.
3. Copy the code block below and paste it exactly as it is into the **URL / Address** field.
4. Save it. Click the bookmark on any webpage to launch the controller!

## 🎮 Controls Map
* **Left Stick**: `W` `A` `S` `D`
* **Right Stick**: Virtual Mouse Pointer
* **D-Pad**: `Up` `Down` `Left` `Right`
* **✖ (Cross)**: Mouse Click (or `Space` if cursor is hidden)
* **● (Circle)**: `Escape`
* **■ (Square)**: `Enter`
* **▲ (Triangle)**: `Shift`
